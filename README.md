## 🔧 Instalación
 1. Clonar el repositorio:
  ```
  $ git clone [repositorio]  
  ``` 
 2. Instalar las dependencias:
```
$ npm install
 ```
 3. Configura el archivo ```.env ``` con tu backend o puedes usar la que tenemos desplegada. 

 4. Inicializa el servidor de desarrollo:
```
$ npm run dev
```
